# 日本メーカーフィルター（自分用）

amazon.co.jp の検索結果で、**日本メーカーの製品に🇯🇵バッジ**を付け、それ以外を**薄く表示**するTampermonkeyユーザースクリプトです。

## 導入手順

1. ブラウザ（Chrome / Edge）に拡張 **Tampermonkey** を入れる（無料）
2. Tampermonkey のアイコン → 「新規スクリプトを作成」
3. 表示されたエディタの中身を全部消し、`amazon-jp-maker-filter.user.js` の中身を貼り付ける
4. `Ctrl + S` で保存
5. **すでに開いている amazon.co.jp のタブは `F5` で再読み込みする**（保存前から開いていたタブには反映されないため）
6. amazon.co.jp で何か検索すると、日本メーカー品にバッジが付く

正しく動いていれば、Amazonのページ上で Tampermonkey のツールバーアイコンに小さな「1」が付き、画面右下にON/OFFボタンが出ます。

画面右下の「🇯🇵 日本メーカー強調：ON/OFF」ボタンで、その場で効果を切り替えられます。

## 設定の変え方（スクリプト冒頭の「設定」欄）

- `MODE = 'badge'` … 日本メーカーにバッジ＋それ以外を薄く表示（初期値）
- `MODE = 'hide'` … 日本メーカー以外を検索結果から消す
- `DIM_OPACITY` … 薄く表示するときの濃さ（0〜1）

## メーカーの追加・削除

`JP_MAKERS` の配列に1行足すだけです。

```js
{ jp: ['カナ・漢字表記'], en: ['英字表記'] },
```

- `jp`（カナ・漢字）は**部分一致**で判定します。
- `en`（英字）は**独立した単語として現れたときだけ**判定します（`connect` の中の `nec` 等には誤反応しません）。
- `mouse` `final` などのありふれた英単語は誤検出が多いため、英字照合には入れていません（カナ表記で拾います）。

## 仕組みと限界（重要）

このツールは **商品タイトルに含まれるメーカー名**で日本メーカーかどうかを判定します。Amazonには信頼できる「製造国」データが無いため、これが現実的な最善手です。そのため以下の限界があります。

- **「日本メーカー」であって「日本製（Made in Japan）」ではない。** 例：ソニー製でも製造は中国、ということは普通にあります。
- **リストに無いメーカーは取りこぼす（見落とし）。** 例：AVIOT は日本のオーディオブランドですが、初期リストに無ければ拾えません。気づいたら `JP_MAKERS` に足してください。
- **短い英字名は誤検出の可能性。** `TIGER`（タイガー魔法瓶）が英単語の "tiger" に反応する等。気になる項目は英字照合から外せます。

## 動作確認の記録

`ワイヤレスイヤホン` の検索結果（60件）で検証：

- 日本メーカー判定 9件（ソニー / JVCケンウッド・Victor / オーディオテクニカ / エレコム）→ すべて実在の日本メーカーで正解
- 除外 51件（Anker=中国 / JBL=米 / Marshall=英 など）→ 正しく除外
- 見落とし：AVIOT（日本ブランド）はリスト未登録のため未検出（限界の実例）
